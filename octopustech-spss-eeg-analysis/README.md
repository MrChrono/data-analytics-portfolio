# OctopusTech SPSS Test Assignment: EEG Data Analysis

## Описание проекта

Проект выполнен в рамках тестового задания OctopusTech.

Цель работы — провести статистический анализ EEG-данных в контексте нейромаркетингового исследования: проверить, изменяется ли мощность EEG-ритмов при предъявлении визуальных стимулов-упаковок по сравнению с baseline-состоянием, а также подготовить аналитическую интерпретацию результатов.

Проект демонстрирует работу с исследовательскими данными, проверку статистических гипотез, анализ связанных измерений, интерпретацию результатов и оформление выводов для прикладного маркетингового контекста.

## Контекст задачи

В исследовании анализировались EEG-показатели участников при предъявлении визуальных стимулов — упаковок товаров.

Основной фокус анализа — сравнение baseline-состояния и реакции на стимулы по трём частотным диапазонам:

- alpha;
- beta;
- theta.

Дополнительно рассматривались различия между отдельными стимулами, сила суммарного эффекта, устойчивость эффекта и возможные профили нейрофизиологической реакции на упаковки.

## Данные

В тестовом задании рассматривались:

- 18 участников;
- 17 стимулов;
- 3 товарные группы: макароны, мука, хлопья;
- baseline-значения и значения во время предъявления стимулов;
- показатели EEG-активности по частотным диапазонам alpha, beta и theta.

Особенность данных — неполная структура наблюдений: не для всех участников были доступны данные по каждому стимулу. Анализ проводился на доступных парных наблюдениях без заполнения пропусков.

## Основные гипотезы

Основная статистическая гипотеза:

- H0: статистически значимые различия между baseline и stimulus по мощности alpha-, beta- и theta-ритмов отсутствуют.

Альтернативные гипотезы:

- H1a: для alpha-ритма есть статистически значимые различия между baseline и stimulus;
- H1b: для beta-ритма есть статистически значимые различия между baseline и stimulus;
- H1c: для theta-ритма есть статистически значимые различия между baseline и stimulus.

Дополнительно проверялась исследовательская гипотеза:

- H2: стимулы с более сильным эффектом характеризуются меньшей вариативностью реакции.

## Что было сделано

### 1. Подготовка данных

- изучена структура исходного SPSS-файла;
- выделены baseline- и stimulus-показатели;
- для каждого сочетания участник–стимул рассчитаны delta-показатели: `stimulus − baseline`;
- подготовлена горизонтальная таблица EEG-показателей по стимулам;
- проверена неполнота данных и ограничения анализа.

### 2. Описательная статистика

- рассчитаны средние значения delta по alpha, beta и theta;
- оценены минимальные и максимальные значения;
- рассчитаны стандартные отклонения;
- описаны различия между стимулами на уровне тенденций.

### 3. Проверка общего эффекта стимулов

Для оценки общего эффекта предъявления упаковок были рассчитаны средние delta-показатели по участнику:

- `mean_d_alpha`;
- `mean_d_beta`;
- `mean_d_theta`.

Затем была проведена проверка отличия средних delta от нуля с помощью одновыборочного t-критерия.

### 4. Сравнение стимулов между собой

Для проверки различий между 17 стимулами использовался критерий Фридмана отдельно для alpha, beta и theta.

Поскольку SPSS для критерия Фридмана использует только полные случаи, анализ между стимулами фактически опирался на 8 участников с полными данными по всем 17 стимулам. Это ограничение отдельно учтено при интерпретации результатов.

### 5. Анализ устойчивости

Дополнительно была проверена устойчивость результатов при исключении участника без eye-tracking-данных. Повторный анализ показал, что основные выводы сохраняются.

### 6. Описательная оценка силы эффекта

Для каждого стимула были рассчитаны:

- средние изменения по alpha, beta и theta;
- суммарная сила эффекта: `|alpha| + |beta| + |theta|`;
- межиндивидуальная вариативность;
- индекс стабильности эффекта: `|M| / SD`.

### 7. Кластерный анализ и бизнес-интерпретация

Стимулы были сгруппированы в три предварительных профиля нейрофизиологической реакции:

1. низкая вовлечённость / подавление;
2. интенсивная переработка;
3. внимание + когнитивный контроль.

Эти профили интерпретировались как предварительные гипотезы для последующего маркетингового и поведенческого тестирования.

## Основные результаты

### Общий эффект

На уровне общей выборки статистически значимого эффекта предъявления упаковок не обнаружено ни для одного из трёх ритмов:

- theta: p = 0.653;
- alpha: p = 0.660;
- beta: p = 0.897.

Это означает, что усреднённый по выборке EEG-эффект предъявления упаковок оказался слабым и статистически незначимым.

### Различия между стимулами

Критерий Фридмана не выявил статистически значимых различий между 17 стимулами:

- theta: p = 0.671;
- alpha: p = 0.439;
- beta: p = 0.465.

Результат следует интерпретировать осторожно, так как анализ опирался только на полные случаи.

### Устойчивость результатов

После исключения участника без eye-tracking-данных выводы сохранились:

- theta: p = 0.745;
- alpha: p = 0.485;
- beta: p = 0.283.

### Описательные тенденции

На описательном уровне были выявлены стимулы с наиболее выраженным суммарным EEG-откликом:

- s5 — Шебекинские, макароны;
- s7 — Каждый день, мука;
- s17 — Ясно солнышко, хлопья;
- s12 — Каждый день, хлопья;
- s4 — Роллтон, макароны.

Минимальный суммарный отклик показали:

- s1 — Каждый день, макароны;
- s15 — Русский продукт, хлопья;
- s10 — Увелка, мука.

### Гипотеза о связи силы эффекта и вариативности

Для theta-ритма была обнаружена умеренная отрицательная статистически значимая связь между силой эффекта и вариативностью:

- Spearman rho = -0.569;
- p = 0.017.

Для alpha и beta аналогичная связь не подтвердилась.

## Аналитический вывод

Исследование не выявило статистически значимого общего эффекта упаковок на уровне всей выборки и не подтвердило статистически значимых различий между отдельными стимулами. При этом описательный анализ показал различия в профилях нейрофизиологической реакции, выделил стимулы с более сильным суммарным откликом и позволил сформулировать практические гипотезы для дальнейшего маркетингового тестирования.

Главная ценность проекта — не в выборе окончательного «победителя», а в демонстрации аналитического подхода: от постановки гипотез и проверки данных до осторожной интерпретации результатов с учётом ограничений выборки.
