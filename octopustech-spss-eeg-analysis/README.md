# OctopusTech SPSS Test Assignment: EEG Data Analysis

## Описание проекта

Проект выполнен в рамках тестового задания OctopusTech.

Цель работы — провести статистический анализ EEG-данных в контексте нейромаркетингового исследования: проверить гипотезы о различиях показателей мозговой активности между baseline-состоянием и реакцией на стимулы, а также подготовить аналитические выводы по результатам статистических тестов.

Проект демонстрирует работу с исследовательскими данными, статистическую проверку гипотез, интерпретацию результатов и оформление аналитического отчёта.

## Контекст задачи

В исследовании анализируются EEG-показатели участников при предъявлении стимулов. Основной фокус — сравнение активности в разных частотных диапазонах и оценка того, отличаются ли показатели во время стимулов от baseline.

В работе используются нормализованные показатели:

- alpha;
- beta;
- theta.

## Данные

В тестовом задании рассматривались данные:

- 18 участников;
- 17 стимулов;
- показатели EEG-активности по частотным диапазонам;
- baseline-значения и значения во время предъявления стимулов.

## Основные аналитические задачи

В рамках проекта нужно было:

- подготовить данные для статистического анализа;
- сформулировать проверяемые гипотезы;
- сравнить baseline и показатели во время стимулов;
- проверить статистическую значимость различий;
- сравнить реакции на разные стимулы;
- интерпретировать результаты в формате, понятном для исследовательского и бизнес-контекста.

## Используемые методы

В проекте применялись:

- описательная статистика;
- сравнение baseline и stimulus-показателей;
- t-test для проверки различий между состояниями;
- Friedman test для сравнения связанных измерений по нескольким стимулам;
- анализ статистической значимости;
- визуализация и табличное представление результатов.

## Используемые инструменты

- IBM SPSS Statistics;
- SPSS syntax;
- Excel / табличная подготовка данных;
- статистический анализ;
- аналитический отчёт.

## Что было сделано

### 1. Подготовка данных

- изучена структура исходных данных;
- определены ключевые переменные для анализа;
- выделены baseline- и stimulus-показатели;
- подготовлены данные для статистической проверки гипотез;
- проверена корректность формата данных для анализа в SPSS.

### 2. Формулировка гипотез

Были сформулированы статистические гипотезы о различиях EEG-показателей:

- между baseline и предъявлением стимулов;
- между разными типами стимулов;
- по отдельным частотным диапазонам alpha, beta и theta.

### 3. Статистический анализ

- рассчитаны описательные показатели;
- проведены сравнения baseline и stimulus-состояний;
- выполнены статистические тесты;
- определены различия, требующие содержательной интерпретации;
- результаты подготовлены для включения в аналитический отчёт.

### 4. Интерпретация результатов

По итогам анализа были подготовлены выводы о том:

- какие EEG-показатели изменяются относительно baseline;
- какие стимулы отличаются по выраженности реакции;
- какие результаты можно считать статистически значимыми;
- какие ограничения нужно учитывать при интерпретации из-за размера выборки и структуры данных.
