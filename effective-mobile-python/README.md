# Effective Mobile Python Test

## Описание

В этом проекте собраны решения тестовых задач на Python.

Задачи направлены на проверку базовых алгоритмических навыков: работа со строками, массивами, числами, структурами данных и оценкой сложности решений.

## Задачи

1. Проверка изоморфности строк.
2. Поиск пропущенного числа в последовательности.
3. Разложение числа на простые множители.

## Используемые инструменты

- Python 3
- dict
- set
- list
- циклы и условия
- оценка временной и пространственной сложности

## Что сделано

- реализованы решения для каждой задачи;
- добавлены проверки базовых и граничных случаев;
- описана логика работы алгоритмов;
- указана оценка сложности решений.

## Файлы проекта

- `isomorphic_strings.py` — проверка изоморфности строк;
- `missing_number.py` — поиск пропущенного числа;
- `prime_factors.py` — разложение числа на простые множители.

## Вывод

Проект демонстрирует базовое владение Python, умение формализовать задачу, выбирать подходящие структуры данных и проверять корректность решения.
