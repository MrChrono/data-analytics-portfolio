# Effective Mobile Data Analyst Test Assignment

## Описание проекта

Проект выполнен в рамках тестового задания на позицию аналитика данных.

Здесь собраны решения по теории вероятностей, Python, SQL, статистике, A/B-тестам и базовым ML-метрикам.

Проект показывает не только готовые ответы, но и ход рассуждения: как формализуется задача, почему выбирается конкретный метод, как проверяется логика решения и как объясняется результат.

## Структура тестового задания

Тестовое состояло из нескольких блоков:

1. Теория вероятностей и логика.
2. Python.
3. SQL.
4. Статистика и A/B-тесты.
5. ML Base.

## Что было сделано

### 1. Теория вероятностей и логика

Решены задачи на:

- математическое ожидание количества уникальных объектов;
- линейность математического ожидания;
- метод индикаторов;
- вероятность события на разных временных интервалах.

### 2. Python

Реализованы функции:

- `is_isomorphic()` - проверка строк на изоморфность;
- `missing_number()` - поиск пропущенного числа в последовательности;
- `prime_factors()` - разложение числа на простые множители.

Для каждой задачи описаны:

- логика решения;
- используемые структуры данных;
- временная сложность;
- дополнительная память.

### 3. SQL

Решены задачи на:

- построение рейтинга через оконную функцию `RANK()`;
- понимание диапазона строк после `FULL JOIN`;
- выбор клиентов с суммой покупок меньше заданного порога;
- использование `JOIN`, `WHERE`, `GROUP BY`, `HAVING`;
- объяснение бизнес-логики запроса.

### 4. Статистика и A/B-тесты

Разобраны задачи на:

- интерпретацию результата A/B-теста;
- репрезентативность и осторожное масштабирование эксперимента;
- понимание p-value;
- применимость t-test при большом размере выборки и ненормальном распределении.

### 5. ML Base

Решены задачи на:

- интерпретацию ROC-AUC;
- выбор модели с ROC-AUC ниже 0.5 через инверсию предсказаний;
- ручной расчёт ROC-AUC;
- ручной расчёт корреляции Пирсона;
- различение корреляции и причинно-следственной связи.
