## Блок 1. Теория вероятностей и логика

### Задача 1. Фермер

**Условие:** есть 6 разных видов животных. Каждый раз фермер видит одно случайное животное. За день он заходит в сарай 6 раз. Нужно найти математическое ожидание количества разных видов животных, которые он увидит.

### Решение

Для каждого вида животного вводим индикатор:

```text
Ii = 1, если i-й вид был замечен хотя бы один раз
Ii = 0, если i-й вид не был замечен
```

Количество разных увиденных видов:

```text
X = I1 + I2 + I3 + I4 + I5 + I6
```

По линейности математического ожидания:

```text
E(X) = E(I1) + E(I2) + ... + E(I6)
```

Для индикатора математическое ожидание равно вероятности события.

Вероятность не увидеть конкретный вид за один заход:

```text
5 / 6
```

Вероятность не увидеть конкретный вид за 6 заходов:

```text
(5 / 6)^6
```

Вероятность увидеть конкретный вид хотя бы один раз:

```text
1 - (5 / 6)^6
```

Итог:

```text
E(X) = 6 * (1 - (5 / 6)^6) ≈ 3.99
```

**Ответ:** 3.99.

### Задача 2. Кулинарное соревнование

**Условие:** в конкурсе 80 шеф-поваров с уникальными уровнями мастерства. Участники дважды случайно распределяются по парам. В каждой паре выигрывает более сильный. Нужно найти математическое ожидание числа участников, выигравших оба этапа.

### Решение

Ранжируем участников от слабейшего к сильнейшему. Пусть `k` - количество участников, которые слабее конкретного шефа.

Тогда вероятность выиграть один этап:

```text
P = k / 79
```

Вероятность выиграть оба этапа:

```text
P = (k / 79)^2
```

Число победителей можно представить как сумму индикаторов:

```text
X = I1 + I2 + ... + I80
```

По линейности математического ожидания:

```text
E(X) = (0/79)^2 + (1/79)^2 + ... + (79/79)^2
```

Используем формулу суммы квадратов:

```text
1^2 + 2^2 + ... + n^2 = n(n + 1)(2n + 1) / 6
```

Для `n = 79`:

```text
1^2 + 2^2 + ... + 79^2 = 79 * 80 * 159 / 6 = 167480
```

Итог:

```text
E(X) = 167480 / 79^2 ≈ 26.83
```

**Ответ:** 26.8.

### Задача 3. Одинокая дорога

**Условие:** вероятность появления автомобиля за 30 минут равна 0.95. Нужно найти вероятность появления автомобиля за 10 минут и за 27 минут.

### Решение

Вероятность отсутствия автомобиля за 30 минут:

```text
1 - 0.95 = 0.05
```

Пусть вероятность отсутствия автомобиля за 10 минут равна `x`. Тогда:

```text
x^3 = 0.05
x = 0.05^(1/3)
```

Вероятность появления автомобиля за 10 минут:

```text
1 - 0.05^(1/3) ≈ 0.6316
```

Для 27 минут:

```text
P(no car for 27 min) = 0.05^(27/30)
P(car for 27 min) = 1 - 0.05^(27/30) ≈ 0.933
```

**Ответ:** 63.2%; 93.3%.

## Блок 4. Статистика и A/B-тесты

### Задача 1. Масштабирование A/B-теста

**Суть задачи:** A/B-тест в небольшом городе показал значимое снижение времени доставки. Нужно решить, можно ли сразу масштабировать изменение на всю страну.

### Решение

Статистическая значимость показывает, что эффект наблюдается внутри конкретной выборки и условий эксперимента. Но она не гарантирует, что эффект воспроизведётся в других условиях.

На результат могли повлиять особенности города:

- плотность застройки;
- расстояния между точками доставки;
- дорожная инфраструктура;
- доступность курьеров;
- особенности транспорта;
- плотность заказов;
- поведение клиентов.

Поэтому результат нельзя автоматически распространять на всю страну.

Корректные варианты:

```text
3. Тест репрезентативен относительно своей генеральной совокупности: таких же небольших городов.
4. Эффект подтвердил потенциал идеи, но перед масштабированием нужно провести эксперименты в других городах.
```

**Ответ:** 3 и 4.

### Задача 2. p-value

**Вопрос:** что такое p-value?

### Решение

p-value - это вероятность получить наблюдаемый результат или ещё более экстремальный результат при условии, что нулевая гипотеза верна.

Чем меньше p-value, тем хуже данные согласуются с предположением об отсутствии эффекта.

**Ответ:** вариант 2.

### Задача 3. Применимость t-test при логнормальном распределении

**Условие:** нужно сравнить среднее число уникальных покупок на пользователя. Выборка очень большая - больше 5 млн наблюдений в группах теста и контроля. Значительных выбросов нет. Распределение метрики логнормальное.

### Решение

t-критерий Стьюдента проверяет гипотезу о различии средних значений.

Несмотря на ненормальность исходного распределения, размер выборки очень большой. Согласно Центральной предельной теореме, при большом размере выборки распределение выборочного среднего стремится к нормальному.

Отсутствие значительных выбросов дополнительно снижает риск нарушения предпосылок.

**Ответ:** t-критерий применять можно.

## Блок 5. ML Base

### Задача 1. Модель с ROC-AUC = 0.1

**Условие:** есть две модели: одна с ROC-AUC = 0.7, другая с ROC-AUC = 0.1. Нужно выбрать модель для дальнейшей работы.

### Решение

ROC-AUC ниже 0.5 означает, что модель ранжирует объекты хуже случайного угадывания. Но если ROC-AUC близок к 0, модель может быть полезной: она систематически ошибается в противоположную сторону.

Если инвертировать направление предсказаний модели с ROC-AUC = 0.1, можно получить модель с ROC-AUC около 0.9.

**Ответ:** выбрать вторую модель и инвертировать направление её предсказаний.

### Задача 2. Ручной расчёт ROC-AUC

### Решение

ROC-AUC можно интерпретировать как вероятность того, что случайно выбранный объект положительного класса получит более высокий скор, чем случайно выбранный объект отрицательного класса.

Количество объектов класса 1:

```text
7
```

Количество объектов класса 0:

```text
8
```

Общее количество пар:

```text
7 * 8 = 56
```

Считаем пары, где объект класса 1 получил более высокий скор, чем объект класса 0:

```text
0.95 -> 8 правильных пар
0.85 -> 7 правильных пар
0.75 -> 6 правильных пар
0.70 -> 6 правильных пар
0.65 -> 6 правильных пар
0.60 -> 6 правильных пар
0.40 -> 3 правильные пары
```

Всего правильных пар:

```text
8 + 7 + 6 + 6 + 6 + 6 + 3 = 42
```

ROC-AUC:

```text
42 / 56 = 0.75
```

**Ответ:** ROC-AUC = 0.75.

### Задача 3. Корреляция Пирсона

**Условие:** нужно рассчитать линейную корреляцию между количеством чашек кофе в день экзамена и итоговым баллом за экзамен.

### Решение

Среднее по X:

```text
X_mean = 2.3
```

Среднее по Y:

```text
Y_mean = 73.2
```

Сумма произведений отклонений:

```text
sum((X - X_mean) * (Y - Y_mean)) = -93.60
```

Сумма квадратов отклонений для X:

```text
sum((X - X_mean)^2) = 8.10
```

Сумма квадратов отклонений для Y:

```text
sum((Y - Y_mean)^2) = 1499.60
```

Формула корреляции Пирсона:

```text
r = sum((X - X_mean) * (Y - Y_mean)) / sqrt(sum((X - X_mean)^2) * sum((Y - Y_mean)^2))
```

Итог:

```text
r = -93.60 / sqrt(8.10 * 1499.60) ≈ -0.85
```

**Ответ:** корреляция Пирсона ≈ -0.85.

### Интерпретация

Наблюдается сильная отрицательная линейная связь: чем больше кофе выпивали студенты, тем ниже, как правило, был результат экзамена.

Но корреляция не доказывает причинно-следственную связь. Возможны скрытые факторы: уровень подготовки, стресс, усталость, недостаток сна. Они могут влиять одновременно и на количество кофе, и на результат экзамена.
